# Spatial Clustering Methods for Imaging-Based Spatial Transcriptomics

This repository contains the code used in my master's dissertation to apply and compare different spatial clustering methods to imaging-based spatial transcriptomics data from mouse liver tissue.

## Repository Structure

```text
masterdissertation/
├── banksy_py/
├── SpaceFlow/
├── cellcharter/
├── napari-sparrow/
├── Final_analysis_A1-1/
└── Final_analysis_A1-2/
```

## Spatial Clustering Pipelines

### `banksy_py`

This folder contains the BANKSY source code obtained from the original BANKSY GitHub repository.

The Jupyter notebooks used to run the BANKSY pipeline on the liver spatial transcriptomics datasets are also included in this folder.

### `SpaceFlow`

This folder contains the SpaceFlow source code obtained from the original SpaceFlow GitHub repository.

The Jupyter notebooks used to run the SpaceFlow pipeline are included in this folder.

### `cellcharter`

This folder contains the CellCharter source code obtained from the original CellCharter GitHub repository.

The Jupyter notebooks used to run the CellCharter pipeline are included in this folder.

### `napari-sparrow`

The napari-spparow repository contains a large number of source code files.

Therefore, only the Jupyter Notebook (.ipynb) files used for data preprocessing and analysis in this dissertation are included in this folder, rather than the complete source code from the original repository.

## Final Analysis

### `Final_analysis_A1-1`

This folder contains the final downstream analysis for the A1-1 liver tissue section.

### `Final_analysis_A1-2`

This folder contains the final downstream analysis for the A1-2 liver tissue section.

For both datasets, the clustering results generated by the following methods were combined:

* Leiden clustering
* BANKSY
* SpaceFlow
* CellCharter

The integrated downstream analyses were performed in the BANKSY Conda environment.

The final analysis notebooks include:

* integration of clustering results from all pipelines;
* comparison of spatial clustering methods;
* cell-type composition analysis;
* adjusted mutual information analysis;
* zonation-score analysis;
* variance-based zonation purity analysis; and
* generation of the final figures and summary results.

## Note

The original source code for SPParOW, BANKSY, SpaceFlow, and CellCharter was obtained from their respective GitHub repositories. The analysis notebooks included in this repository were adapted for the datasets and research objectives of this master's dissertation.
